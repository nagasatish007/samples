# Returns Checklist

Use this guide when a customer asks about edge cases.

- If the exact model is not in the catalog, map it to the nearest supported category before calling tools.
- For damaged items, clearly state that inspection may affect refund timing.
- For opened accessories, note that original components should be included whenever possible.
- For warranty questions, combine the return policy with product information so the customer can distinguish return windows from warranty coverage.
- If you are missing a required fact, say so directly and recommend contacting technical support.
